# Workshop UI Library

Reusable enterprise SaaS UI primitives for Workshop.

## Rules
- Use these primitives before creating page-specific styles.
- Keep density compact (`32-36px` controls, `13px` body copy).
- Keep radii tight (`6-8px` default).
- Keep states tokenized (focus, hover, disabled).

## Components
- `alert`
- `badge`
- `button`
- `card`
- `field`
- `input`
- `label`
- `select`
- `separator`
- `table`
- `textarea`

See `docs/design-system/WORKSHOP-UI-STANDARDS.md` for full standards.
